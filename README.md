## **Code of AI-Abstinence**

I, **Chad**, hereby solemnly swear (or affirm) that every single byte of code in this repository will be written by my own mortal hand, fueled only by the raw power of my brain, caffeine, and occasional snacks. 

### I pledge:
- To resist the siren song of AI assistance, no matter how tempting the ChatGPT tab may look at 2 AM.
- To solve every Advent of Code challenge with my own wits, sweat, and probably more Googling than I'd like to admit.
- That if an AI even *whispers* a hint in my ear, I will instantly disqualify myself from counting that day's solution as legitimate. (And we don’t want that. Nobody wants that.)

### Exceptions (because let's be real):
- AI may suggest README formatting or correct typos, but no AI-generated logic, algorithms, or clever regex is allowed to infiltrate this sacred codebase.  
- Debugging advice from humans who used AI? Dubiously ethical, but we'll cross that bridge when we get to it.  

Let this repository stand as proof that I can conquer Advent of Code with grit, determination, and absolutely no robotic shortcuts. If I fail, it's all on me—and I take full responsibility for my stubborn humanity.
